# FakeStore E-Commerce App

## 📌 Project Overview
This project is a React-based e-commerce application built using FakeStoreAPI.  
It allows users to view, create, update, and delete products.

---

## 🚀 Features

### 1. View Products
- Fetch products from FakeStoreAPI
- Display products using reusable ProductCard components

### 2. Product Details
- View individual product details using dynamic routing
- Uses `useParams` to retrieve product ID

### 3. Add Product
- Form with controlled inputs
- Sends POST request to API

### 4. Edit Product
- Pre-filled form using existing product data
- Updates product via PUT request

### 5. Delete Product
- Confirmation prompt before deletion
- Sends DELETE request
- Redirects to product list

---

## 🛠 Tech Stack

- React
- React Router
- Axios
- FakeStoreAPI
